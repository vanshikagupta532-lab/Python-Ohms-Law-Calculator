# Python-Ohms-Law-Calculator
A Python project that calculates voltage, current and resistance using Ohm's Law.

## Description

A Python mini project that calculates Voltage, Current and Resistance using Ohm's Law.

## Formula Used

V = I × R

Where:

* V = Voltage
* I = Current
* R = Resistance

## Features

* Calculate Voltage
* Calculate Current
* Calculate Resistance
* Simple Menu Driven Program
